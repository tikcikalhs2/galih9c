Nama: Galih Gumilar Yogaswara

Kelas: 9B

Proyek: Website Prototipe Toko Online


Latar Belakang:

Website ini dibuat sebagai proyek tugas akhir mata pelajaran Teknologi Informasi dan Komunikasi (TIK). Tujuan dari pembuatan website ini adalah untuk mempraktikkan kemampuan dasar dalam membuat halaman web menggunakan bahasa HTML dan CSS secara manual tanpa bantuan platform otomatis.

Tema website yang dipilih adalah prototipe toko online bernama CenterCOM, yang berfokus pada produk-produk jaringan dan komputer seperti kabel, switch, dan perangkat pendukung lainnya.

Melalui proyek ini, pembuat website mempelajari cara mengatur layout halaman, menampilkan gambar produk, menambahkan navigasi antarhalaman, serta memberikan tampilan profesional yang mudah digunakan.

Website ini tidak hanya menjadi latihan teknis, tetapi juga sebagai bentuk kreativitas dalam desain antarmuka, dengan menggabungkan warna-warna tegas seperti merah dan hitam untuk memberikan kesan teknologi modern.

Diharapkan hasil akhir dari proyek ini dapat menjadi inspirasi bagi siswa lain untuk terus mengembangkan kemampuan digital dan memahami pentingnya perancangan web di era modern.

Terima kasih telah mengunjungi halaman profil ini.
Segala desain, struktur, dan konten website dibuat sepenuhnya oleh siswa dengan bimbingan guru TIK.
Website ini merupakan bentuk nyata pembelajaran digital yang kreatif dan inovatif.
